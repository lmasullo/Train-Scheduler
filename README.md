# Project Title

Train Scheduler

> Description:

Need to know when the next train is coming in? This is your app! Enter the Train Name, Destination, Arrival Time, and Frequency and it will keep track of all the arrivals.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

None

### Installing

Clone this repository and open the index.html file in your browser of choice, that's it!

## Deployment

To deploy this on a live system, just upload the included files to your server.
